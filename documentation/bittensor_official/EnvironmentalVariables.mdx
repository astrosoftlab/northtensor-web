# Environmental Variables

All the environment variables have the same structure
BT*<object name>*<parameter name>

i.e. 

- BT_SUBTENSOR_NETWORK
- BT_WALLET_NAME
- BT_DATASET_BATCH_SIZE

To change an environmental variable:

export BT <object name><parameter name>

i.e. 

- export BT_AXON_PORT = 6080

changes default port to 6080
